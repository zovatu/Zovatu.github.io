What is Zovatu?

Zovatu is a powerful and user-friendly product generator tool that helps you easily create attractive HTML code for your e-commerce products.

HTML Code Generation

Automatically create beautiful HTML code for products

Live Preview

See your product appearance in real-time

WhatsApp Order

Direct order sending facility to WhatsApp

Data Storage

Save and manage your product information

Responsive Design

Perfect display on all devices

Multilingual Support

Use in Bengali and English languages

Tip: Zovatu is completely free and requires no registration. Just login and start using!

How to Get Started

1

Login

First go to index.html page and login with your username and password. Default user: admin, password: admin123

2

Enter Dashboard

After successful login you will automatically go to the dashboard where you will find all tools

3

Select Language

Select your preferred language (Bengali/English) from the sidebar menu

4

Field Customize (Optional)

Turn on/off form fields as needed from Field Customize section

5

Start Creating Products

Now you can generate HTML code with product information

Important: Before first use, make sure to set your required fields in Field Customize section.

Field Customization

Field customization feature gives you the ability to control the form fields displayed on the dashboard. Through this you can show only the necessary fields.

How to Use

1

Go to Field Customize

Click on "Field Customize" from the sidebar menu

2

Select Fields

Check the fields you need, uncheck the unnecessary ones

3

View Preview

See active fields in the preview section below

4

Update

Click "Update" button to save settings

Field Types

Required Fields

These will always be shown and cannot be turned off

Optional Fields

These can be turned on/off according to your needs

Remember: After changing field settings, go back to dashboard to see the changes.

Advanced Features

WhatsApp Order System

A special feature of Zovatu is the automatic WhatsApp order system. When customer clicks "Order Now" button, an order message will be automatically created and sent to WhatsApp.

Order Message Format: New Order Product: [Product Name] Price: à§³[Price] Want to order

Video Integration

If you provide YouTube video link, the video will be automatically embedded in the product page. This helps customers get a better idea about the product.

Responsive Design

All product pages created with Zovatu are fully responsive. This means:

Desktop

Perfect display on large screens

Tablet

Optimized for medium size screens

Mobile

Touch-friendly on small screens

Custom Styling

The generated HTML code includes CSS styles that you can customize as needed:

/* Change product card color */ .product-card { background: #your-color; border: 2px solid #your-border-color; } /* Button style */ .order-btn { background: linear-gradient(45deg, #color1, #color2); }

Theme Download

From the "G9Theme" box in the sidebar menu you can download blogger theme. This theme is compatible with Zovatu and optimized for displaying products on your blog.

Warning: Be careful when adding custom CSS so that responsive design is not damaged.
